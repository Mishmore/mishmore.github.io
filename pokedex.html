Pokédex

Brief

"¡Pokédex es un sitio web que aloja todo lo que siempre quisite saber de tus pokemones favoritos! Sólo escribe el nombre del pokemóm de tu interés y prepárate para atraparlo.

Desarrollé "¡Pokédex durante el Sprint 5 de jQuery, Ajax, Promises y APIs, en el Bootcamp de Laboratoria. El reto fue utilizar el API de Pokeapi para obtener toda la base de datos de los pokémones y seguir trayendo más información una vez seleccionado el pokémon.

Objetivos

- Crear todos los elementos de la página desde Javascript, utilizando para ello componentes de cada sección de la página.

- Conseguir las imágenes de los pokémones, en coherencia con los identificadores que el API de Pokeapi proporciona.

- Mostrar la información obtenida de un pokémon específico mediante un modal, sin recargar la página.

Proceso

Herramientas utilizadas:

Javascript - HTML5 - CSS - jQuery - Materialize

Aprendizaje

Éste proyecto fue un ejercicio interesante de investigación, pues tuve que leer la documentación del API de Pokeapi, conocer cuáles eran los endpoints útiles para mi sitio web y saber cuál era el momento propicio para realizar los requests. por mi cuenta cómo utilizar el API de Pokeapi y de qué manera podía aplicarlo a mi proyecto. En este proyecto logré interiorizar los conceptos de AJAX y ponerlos en práctica. Además, tuve la oportunidad de realizar validaciones rigurosas, evaluando cada caso que pueda causar algún tipo de conflicto al enviar la data. Logré también entender el flujo de trabajo en una aplicación web que requiera de uso de componentes para una carga más rápida.
