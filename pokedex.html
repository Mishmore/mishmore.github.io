Pokédex

Brief

"¡Pokédex es una app web que aloja todo lo que siempre quisite saber de tus pokemones favoritos! Sólo escribe el nombre del pokemóm de tu interés y prepárate para atraparlo.

Desarrollé "¡Pokédex durante el Sprint 5 de jQuery, Ajax, Promises y APIs, en el Bootcamp de Laboratoria. El reto fue utilizar el API de Pokeapi para obtener toda la base de datos de los pokémones y seguir trayendo más información una vez seleccionado el pokémon.

Objetivos

- Crear todos los elementos de la página desde Javascript, utilizando para ello componentes de cada sección de la página.

- Conseguir las imágenes de los pokémones, en coherencia con los identificadores que el API de Pokeapi proporciona.

- Mostrar la información obtenida de un pokémon específico mediante un modal, sin recargar la página.

Proceso

Herramientas utilizadas:

Javascript - HTML5 - CSS - jQuery - Materialize

Aprendizaje

Éste proyecto fue un ejercicio interesante de investigación, pues tuve que leer la documentación del API de Pokeapi, conocer cuáles eran los endpoints útiles para mi aplicación web y saber cuál era el momento propicio para realizar los requests.

Además, el hecho de practicar las peticiones AJAX, durante todo el desarrollo de la aplicación me permitió comprender cuál era su comportamiento de una manera mucho más práctica y sencilla.
