Pokédex

"¡Pokédex es una app web que aloja todo lo que siempre quisite saber de tus pokemones favoritos! Sólo escribe el nombre del pokemóm de tu interés y prepárate para atraparlo."

Brief

Pokédex es un proyecto desarrollado en el Bootcamp de Laboratoria. El reto fue utilizar el API de Pokeapi para obtener información general de todos los pokémones y lugeo, obtener los datos específicos de uno seleccionado.

Ver código
Ver demo

Para usar el app: Busca el pokémon que desees y luego haz click en la pokebola para visualizar el modal con todos sus datos.

Objetivos

1

Crear todos los elementos de la página mediante componentes desde Javascript.

2

Realizar peticiones al API de Pokeapi para cargar toda la información necesaria.

3

Realizar peticiones para conseguir las imágenes de los pokémones.

4

Mostrar la información obtenida de un pokémon específico mediante un modal.
