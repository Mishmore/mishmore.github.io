Pinterest

"Pinterest es una plataforma que permite a los usuarios crear boards y compartir imágenes. ¿Su misión? Conectar a todos en el mundo a través de cosas que encuentren interesantes."

Brief

El proyecto Pinterest fue desarrollado grupalmente en el Bootcamp de Laboratoria. El principal desafío recrear el board de un usuario real e implementar las principales funcionalidades del sitio.

Ver código en Github

Objetivos

1

Recrear el diseño responsive de la web, apto para dispositivos de escritorio y móviles.

2

Utilizar el API de Pinterest para traer información del board especificado.

3

Automatizar tareas durante el desarrollo con Gulp.

4

Publicar el proyecto en Heroku.

Rol en el proyecto

Configuré Gulp la automatización de tareas y aumentar la productividad del equipo.

Realicé API requests para almacenar data del board y ususario.

Desarollé el prototipado de la sección de navegación y detalle del pin.
