Pinterest

Brief

"Pinterest es una plataforma que permite a los usuarios crear boards temáticos donde pueden crear, guardar y compartir imágenes (pines) de distintas fuentes. ¿Su misión? Conectar a todos en el mundo a través de cosas que encuentren interesantes.

El proyecto Pinterest fue desarrollado en el Bootcamp de Laboratoria. El principal desafío fue recrear un board de un usuario real siguiendo el diseño original de la página.

Objetivos

- Replicar el diseño para desktop y mobile.

- Usar el API de Pinterest para traer información del board del usuario: arabelyuska/web-ui/

- Automatizar tareas con Gulp.

- Publicar el proyecto en Heroku.

Rol en el proyecto

Para garantizar la productividad del equipo en el tiempo disponible, me encargué de realizar las tareas para la actualización automatizada de los archivos de Sass, HTML5 y Javacript en carpeta publics. También adicioné una tarea para recargar el navegador una vez que haya detectado algún cambio en unos de los archivos.

Hice el maquetado de la sección de la barra de navegación para desktop y mobile con el sistema de grillas del Framework Materialize. Tomé en cuenta la visibilidad de sus íconos en ambos casos y el efecto de traslado que tenía en el título del board al deslizar la rueda del mouse hacia abajo.

Realicé el pedido al API para traer la data del usuario y del board. Ésta información está visible en la sección de encabezado, justo debajo de la barra de navegación.

Desarrollé el modal que aparece al hacer click en cada pin del board. Para su diseño y funcionalidad utilicé la estructura de Materialize. Para visualizar la información e imagen que se muestra dentro del pin almacené la data del mismo al momento del click mediante un evento jQuery. Una vez cerrado, creé una función que limpiaba el objeto donde se almacenó las propiedades del pin y de los elementos DOM.

Proceso

Herramientas utilizadas:

Javascript - HTML5 - CSS - jQuery - Materialize - Gulp

Aprendizaje

En este proyecto reforcé mis conocimientos con Gulp, optimizando el proceso de trabajo y aumentando la productividad entre tarea y tarea. De esta manera, logramos llegar a la meta el plazo establecido y cumpliendo todos los requerimientos.
