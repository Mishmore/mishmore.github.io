Pinterest

"Pinterest is a plataform that allows its users create boards and share images. Its mission? Connect everbody in the world through things they found interesting.”

Brief

The Pinterest Project was a team work developed at Laboratoria’s Bootcamp. The main challenge was to recreate a real user board and implement the principal functionalities of the site.

View code
View demo

To use the app: Explore the pins of the board by clicking on each one. It’ll open a modal with information.

Goals

1

Recreate the responsive design, adaptable to desktop and mobile devices.

2

Use the Pinterest API to fetch information from the specified board.

3

Automate tasks during development with Gulp.

4

Publish the project in Heroku.

Project Role

I configured Gulp to automate tasks and increase the productivity of the team.

I developed the prototyping of the navigation section and details of the pin.

I made API requests to store board and user data.
