Yape

"Yape es la nueva aplicación del Banco de Crédito del Perú, con la que se puede realizar transferencias de dinero desde el móvil. Con Yape podrás convertir tus planes en experiencias únicas, sin números de cuenta ni token."

Brief

Esta aplicación fue un proyecto individual en el Bootcamp de Laboratoria. El principal objetivo fue simular el proceso de registro a la app.

Ver código en Github

Objetivos

1

Utilizar componentes para crear la estructura de la app y tener un flujo entre pantallas sin recarga.

2

Validar todos los campos de registro para garantizar un envío de data libre de errores.

3

Realizar de manera efectiva peticiones a la base de datos de la aplicación para enviar la data del registro del usuario.

Aprendizaje

Tuve una perspectiva más amplia de la lógica detrás de una aplicación al recrear el flujo entre pantallas según las decisiones del usuario.

El tipo de restricciones que apliqué en cada validación es fundamental para evitar errores en las peticiones y mantener la base de datos con información útil.

El uso de un sistema de componentes es primordial en aplicaciones móviles, donde el factor de tiempo es a veces determinante para la aceptación y uso de una app.
